# Exit Mode

Exit any active mode (creative, challenger, teaching) and return to context-aware default responses.

## When Invoked

Respond with:
"Exited [mode name] mode — back to default."

If no mode was active:
"No active mode to exit — already in default."

## Natural Language Triggers

Also activate when user says:
- "exit mode"
- "back to normal"
- "default mode"
- "stop [mode name] mode"
- "reset"

## Behavior After Exit

- Resume context-aware signal detection
- Apply ADHD support as usual (TL;DR, thread anchors)
- No forced template structure
- Match response style to each individual request
