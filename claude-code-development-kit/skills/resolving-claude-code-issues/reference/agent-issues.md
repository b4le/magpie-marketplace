# Agent Issues

Troubleshooting Task agents, Explore agents, and other sub-agent problems.

## Agent Not Completing

**Problem**: Task agent runs but doesn't complete

**Solutions**:
1. Check agent has clear task description
2. Verify sufficient context provided
3. Check for errors in agent output
4. Try more specific prompt
5. Break into smaller tasks

## Agent Using Wrong Tools

**Problem**: Agent not using expected tools

**Solutions**:
1. Make tool requirements explicit
2. Provide examples
3. Check allowed-tools restrictions
4. Verify tools are available

## Explore Agent Not Finding Code

**Problem**: Explore agent can't find expected code

**Solutions**:
1. Increase thoroughness level
2. Provide more context
3. Check file paths are correct
4. Verify code actually exists
5. Try different search terms
